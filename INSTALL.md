Please refer to the appropriate Markdown file in the `doc/` directory for your operating system:

| Operating System | Build Guide Link |

| **Windows (MinGW)** | [`doc/build-mingw.md`](doc/build-mingw.md) |
| **OpenBSD** | [`doc/build-openbsd.md`](doc/build-openbsd.md) |

Before starting the build process, ensure you have reviewed the **system dependencies** listed in the respective guide for your platform.
